Setup & Installation

1.  Clone: git clone
    https://github.com/asfiafaiso/periodic-table-project.git

2.  Install dependencies: npm install

3.  Build project: npm run dev

4.  Run locally: Run PHP dev server: php -S localhost:8000

Usage

-   Open localhost:8000 in browser
-   Sign in Google account
-   Click button to change layout
